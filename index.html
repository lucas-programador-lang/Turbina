PROCESSO

Quatro passos, do download ao ganho de desempenho.

Sem conta, sem cadastro, sem app rodando em segundo plano depois. Você baixa, executa, escolhe o que quer aplicar, e pronto.

01

Baixe o script do seu sistema

Um arquivo único: .bat para Windows, .sh para macOS ou Linux.

02

Execute com permissão de administrador

No Windows, clique com o botão direito. No Mac/Linux, use chmod +x e rode no Terminal.

03

Escolha no menu interativo

Cada etapa é opcional e explicada na hora — você decide o que aplicar, nada roda escondido.

04

Reinicie e sinta a diferença

Um log é salvo na mesma pasta, mostrando exatamente o que foi alterado.

ESCOLHA SEU SISTEMA

Um script dedicado para cada plataforma.

Mesmo espírito, comandos nativos de cada sistema — nada de tradução forçada de um SO para outro.

Windows

turbina-windows.bat · Windows 10 / 11
  • Menu com 28 otimizações, cada uma opcional
  • Detecta SSD ou HD e ajusta sozinho
  • Turbo de internet, DNS, limpeza e reparo do sistema
  • Cria ponto de restauração antes de mexer em qualquer coisa
Baixar turbina-windows.bat
chmod não necessário · execute como Administrador

macOS

turbina-mac.sh · Intel & Apple Silicon
  • Limpa cache do sistema e de apps
  • Verifica disco e libera memória RAM inativa
  • Reinicia Finder e Dock sem precisar deslogar
  • Atalho direto para os itens de login
Baixar turbina-mac.sh
chmod +x turbina-mac.sh

Linux

turbina-linux.sh · Distros com systemd
  • Detecta apt, dnf, pacman ou zypper sozinho
  • Roda TRIM no SSD e ajusta o swappiness
  • Lista serviços ativos na inicialização
  • Reduz logs antigos do journalctl
Baixar turbina-linux.sh
chmod +x turbina-linux.sh

DÚVIDAS

Perguntas frequentes

Esses scripts são seguros?

Sim. Todo comando é de código aberto, comentado e usa apenas ferramentas nativas do próprio sistema operacional (Registro do Windows, PowerShell, systemd, etc). Nada é baixado de servidores externos, e ações mais sensíveis (como remover apps ou desativar hibernação) sempre pedem confirmação antes de rodar.

Preciso desinstalar algo depois?

Não. Não existe instalador, serviço em segundo plano ou processo residente. É um único arquivo de script que roda e termina — você pode até apagar o arquivo depois de usar.

E se alguma mudança não me agradar?

No Windows, o script cria um ponto de restauração do sistema antes de qualquer alteração — é só reverter por lá. Nos scripts de Mac e Linux, cada ação é isolada e reversível manualmente (o próprio script explica como desfazer as mais sensíveis, como o swappiness).

Funciona em qualquer versão do sistema?

O script de Windows foi testado em Windows 10 e 11. O de macOS funciona em versões recentes (Intel e Apple Silicon). O de Linux funciona em qualquer distribuição baseada em systemd — Ubuntu, Debian, Fedora, Arch, Mint, Pop!_OS, entre outras.
